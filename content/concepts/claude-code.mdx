## Définition

Claude Code est un outil développé par <Term>Anthropic</Term> qui permet d'interagir avec le modèle <Term>Claude</Term> pour écrire, modifier et gérer du code. Contrairement aux interfaces de chat classiques, Claude Code dispose d'un accès direct au système de fichiers et peut exécuter des commandes.

## Fonctionnalités principales

### Accès au projet

Claude Code peut lire l'arborescence d'un projet, naviguer entre les fichiers, et comprendre la structure globale du codebase. Cette capacité lui permet de proposer des modifications cohérentes avec l'existant.

### Exécution de commandes

L'outil peut lancer des commandes système : installation de <Term>Dépendances</Term>, exécution de tests, opérations Git. Chaque action requiert une validation explicite de l'utilisateur. (Sauf si configuration explicite.)

### Mode <Term>Cloud</Term>

Claude Code Cloud permet d'utiliser l'outil directement depuis un navigateur, sans installation locale. Le code s'exécute sur les serveurs d'<Term>Anthropic</Term> et se synchronise avec <Term>GitHub</Term>. Moins de liberté cependant : pas de skill, de commandes, etc.

### Mode Local

Une version <Term>CLI</Term> existe pour travailler directement sur sa machine. Elle offre plus de contrôle et est la version originelle de Claude Code, et le premier <Term>Agent IA</Term> <Term>CLI</Term>.

## Cas d'usage

- Génération de fonctionnalités complètes
- Refactoring de code existant
- Débogage assisté
- Création de projets from scratch
- Automatisation de tâches répétitives

## Limites

- Consommation de <Term>Token</Term>s proportionnelle à la complexité des tâches (Mais cost insensitive sachant que le prix est à l'abonnement)
- Pas vraiment de limite, c'est vraiment un outil très puissant.
