## Définition

Claude Code est un outil développé par Anthropic qui permet d'interagir avec le modèle Claude pour écrire, modifier et gérer du code. Contrairement aux interfaces de chat classiques, Claude Code dispose d'un accès direct au système de fichiers et peut exécuter des commandes.

## Fonctionnalités principales

### Accès au projet

Claude Code peut lire l'arborescence d'un projet, naviguer entre les fichiers, et comprendre la structure globale du codebase. Cette capacité lui permet de proposer des modifications cohérentes avec l'existant.

### Exécution de commandes

L'outil peut lancer des commandes système : installation de dépendances, exécution de tests, opérations Git. Chaque action requiert une validation explicite de l'utilisateur.

### Mode Cloud

Claude Code Cloud permet d'utiliser l'outil directement depuis un navigateur, sans installation locale. Le code s'exécute sur les serveurs d'Anthropic et se synchronise avec GitHub.

### Mode Local

Une version CLI existe pour travailler directement sur sa machine. Elle offre plus de contrôle et fonctionne hors ligne une fois installée.

## Différences avec les autres assistants de code

| Aspect | Claude Code | Assistants classiques |
|--------|-------------|----------------------|
| Accès fichiers | Direct | Via copier-coller |
| Exécution commandes | Oui | Non |
| Contexte projet | Complet | Limité |
| Modifications | Directes | Suggestions uniquement |

## Cas d'usage

- Génération de fonctionnalités complètes
- Refactoring de code existant
- Débogage assisté
- Création de projets from scratch
- Automatisation de tâches répétitives

## Limites

- Consommation de tokens proportionnelle à la complexité des tâches
- Nécessite une connexion internet en mode cloud
- Le code généré peut nécessiter des ajustements manuels
- Certaines opérations système restent restreintes pour des raisons de sécurité
