# Comprendre les LLM

Les **Large Language Models** (LLM) sont au coeur de la revolution actuelle de l'IA. Mais comment fonctionnent-ils vraiment ?

## Qu'est-ce qu'un LLM ?

Un LLM est un modele d'apprentissage profond entraine sur d'enormes quantites de texte. Il apprend a predire le prochain mot dans une sequence, ce qui lui permet de generer du texte coherent.

### Les concepts cles

1. **Tokenisation** : Le texte est decoupe en tokens (mots ou sous-mots)
2. **Embeddings** : Chaque token est converti en vecteur numerique
3. **Attention** : Le mecanisme qui permet au modele de "se concentrer" sur les parties pertinentes

## Architecture Transformer

L'architecture Transformer, introduite en 2017, est la base de tous les LLM modernes.

```python
# Exemple simplifie d'attention
import torch
import torch.nn.functional as F

def attention(query, key, value):
    scores = torch.matmul(query, key.transpose(-2, -1))
    weights = F.softmax(scores, dim=-1)
    return torch.matmul(weights, value)
```
