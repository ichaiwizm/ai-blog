Parmi les fonctionnalités les plus impressionnantes de Claude Code, l'**extension Chrome** se distingue par sa capacité à prendre le contrôle de votre navigateur pour exécuter des tâches à votre place. Certes, elle n'est pas la plus rapide, mais pour les opérations répétitives et fastidieuses, elle devient un véritable allié.

## Qu'est-ce que l'extension Chrome de Claude Code ?

L'extension Chrome de Claude Code permet à l'<Term>IA</Term> de **contrôler directement votre navigateur**. Elle peut :

- Naviguer sur des sites web
- Remplir des formulaires
- Cliquer sur des boutons
- Lire le contenu des pages
- Effectuer des séquences d'actions complexes

C'est ce qu'Anthropic appelle le **Computer Use** appliqué au navigateur : Claude voit votre écran, comprend l'interface et agit comme vous le feriez.

<Callout type="warning">
L'extension Chrome de Claude Code est **exclusivement disponible pour les abonnés Max** (100$/mois ou 200$/mois). Les abonnés Pro n'y ont pas accès.
</Callout>

## Installation et activation

Pour utiliser l'extension :

1. Installez l'extension depuis le Chrome Web Store
2. Connectez-vous avec votre compte Claude (abonnement Max requis)
3. L'extension apparaît dans la barre d'outils de Chrome

Une fois installée, Claude Code peut interagir avec votre navigateur lorsque vous lui demandez d'effectuer des tâches web.

## Performances : lent mais fiable

Soyons honnêtes : l'extension n'est **pas rapide**. Chaque action prend quelques secondes car Claude doit :

1. Capturer l'état de la page
2. Analyser le contenu visuel
3. Décider de l'action à effectuer
4. Exécuter l'action
5. Vérifier le résultat

Pour une tâche unique, cette lenteur peut sembler frustrante. Mais la vraie valeur de l'extension réside dans l'**automatisation des tâches répétitives** où vous n'avez pas besoin d'être présent.

## Cas d'usage : automatiser la candidature à des offres d'emploi

Voici un exemple concret d'utilisation avancée. J'ai récemment automatisé un workflow complet :

1. **Claude analyse les offres d'emploi** sur différentes plateformes
2. **Il personnalise mon CV** selon chaque offre (compétences mises en avant, mots-clés adaptés)
3. **Il déploie automatiquement** chaque version via <Term>Vercel</Term> <Term>CLI</Term>
4. **Il postule** sur les plateformes avec le lien vers le CV déployé

Le tout sans intervention manuelle. Pendant que Claude travaille, je peux faire autre chose.

```bash
# Exemple de prompt pour lancer l'automatisation
claude "Analyse les 5 dernières offres de développeur React sur LinkedIn,
personnalise mon CV pour chacune en mettant en avant les compétences demandées,
déploie chaque version sur Vercel et postule avec le lien correspondant."
```

## Cas d'usage : supprimer des projets Vercel en masse

Un autre exemple parfait pour illustrer la puissance de l'extension : **la suppression de projets Vercel**.

Si vous avez déjà essayé de supprimer un projet sur Vercel via l'interface web, vous savez à quel point c'est pénible :

1. Aller sur la page du projet
2. Cliquer sur **Settings**
3. Descendre tout en bas de la page
4. Cliquer sur **Delete Project**
5. Copier le nom exact du projet
6. Le coller dans le champ de confirmation
7. Écrire "delete the project" dans un autre champ
8. Confirmer la suppression

**Huit étapes** pour supprimer un seul projet. Et je ne suis même pas sûr que la <Term>CLI</Term> Vercel permette de le faire directement.

<Callout type="info">
À vérifier : la commande `vercel remove` existe mais semble limitée aux déploiements, pas aux projets entiers. L'interface web reste parfois la seule option pour certaines opérations destructives.
</Callout>

J'avais **5 projets à supprimer**. Quarante étapes manuelles, une tâche parfaite pour l'extension.

```bash
claude "Supprime les projets Vercel suivants : projet-test-1, projet-test-2,
projet-test-3, ancien-portfolio, demo-client.
Va sur vercel.com/dashboard, et pour chaque projet,
va dans settings et suis le processus de suppression."
```

Claude a tout fait tout seul. Connexion, navigation, confirmations, le tout sans erreur. Pendant ce temps, je préparais mon café.

## Quand utiliser l'extension Chrome ?

L'extension excelle dans certains scénarios :

| Situation | Recommandé ? |
|-----------|--------------|
| Tâche unique rapide | Non — faites-le vous-même |
| 2-3 répétitions | Limite — dépend de la complexité |
| 5+ répétitions identiques | Oui |
| Tâches avec beaucoup de clics/confirmations | Oui |
| Actions sur plusieurs sites | Oui |
| Scraping léger | Oui |
| Automatisation de workflows web | Oui |

## Limites et précautions

Quelques points à garder en tête :

- **Lenteur** : ne convient pas aux actions urgentes
- **Consommation de tokens** : chaque capture d'écran et analyse consomme des tokens de votre quota
- **Sites protégés** : certains sites détectent et bloquent l'automatisation
- **Authentification** : Claude peut interagir avec vos sessions connectées — soyez prudent avec les informations sensibles

<Alert variant="warning">
Ne demandez jamais à Claude d'effectuer des opérations sensibles comme des transactions bancaires ou la modification de données critiques sans supervision.
</Alert>

## En résumé

L'extension Chrome de Claude Code transforme l'IA en **assistant capable d'agir dans votre navigateur**. Elle n'est pas conçue pour la vitesse mais pour la **délégation de tâches répétitives**.

Les abonnés Max qui l'utilisent découvrent rapidement son potentiel : au lieu de cliquer pendant 20 minutes sur des formulaires identiques, vous décrivez ce que vous voulez et vous passez à autre chose pendant que Claude s'en occupe.

C'est ça, le vrai gain de productivité avec l'IA : non pas aller plus vite, mais **ne plus avoir à le faire du tout**.
